# $divide
Данная функция разделяет заданные числа
### Использование
```php
$divide[...числа]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| числа | Числа, которые будут разделены | Число | Да |  
## Пример(ы)

```javascript
bot.command({
  name: '$divide',
  code: `
$divide[10;5]`
// Возвращает: 2
})
```
